[WEB ACCESS LAYER & NFM-ID BRIDGE]

1. THE NFM-ID BRIDGE (QR LOGIN)
--------------------------------------------------------------------------------
- Browser meminta login -> Generate QR -> HP Scan (Biometrik) -> 
  HP kirim ZK-Proof ke Browser via Mesh -> Login Berhasil.
- Keuntungan: Keamanan HP digabung dengan kenyamanan layar lebar Browser.

2. DECENTRALIZED WEB HOSTING
--------------------------------------------------------------------------------
- File website NFM dipecah (sharding) dan disimpan di jaringan NFM sendiri.
- Mencegah DNS Blocking atau sensor oleh pihak ketiga.

3. WEB-TO-MESH RPC GATEWAY
--------------------------------------------------------------------------------
- Browser berinteraksi dengan Blockchain NFM melalui Edge Gateway.
- Ringkasan data (Explorer) diambil secara cerdas tanpa harus download shard AI.
