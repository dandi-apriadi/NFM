[KOMPONEN KONTROL RESOURCE ADAPTIF]

1. PERFORMANCE MODES (User-Defined)
--------------------------------------------------------------------------------
Node Runner mendukung 4 profil penggunaan daya:
- **QUIET**: Menggunakan < 10% resource. Cocok untuk laptop saat bekerja atau HP 
  di siang hari. Fokus pada stabilitas sistem utama.
- **BALANCED**: Menggunakan 40-60% resource. Keseimbangan antara reward dan 
  kenyamanan penggunaan perangkat.
- **TURBO**: Menggunakan 80-90% resource. Prioritas tinggi untuk NFM, namun tetap 
  menyisakan sedikit ruang untuk sistem operasi.
- **HARDCORE**: [MAX] Menggunakan 100% hardware (Semua Core CPU, Full VRAM, 
  Max NPU). Perangkat didedikasikan sepenuhnya untuk network NFM.

2. DYNAMIC GOVERNOR (Mode Orchestration)
--------------------------------------------------------------------------------
- Sistem akan mengatur `thermal throttling` secara otomatis agar perangkat tidak 
  rusak saat mode Hardcore diaktifkan.
- Jika suhu melewati batas aman, Governor akan menurunkan mode secara otomatis 
  dari Hardcore ke Balanced hingga suhu stabil.

3. REWARD SCALING BY MODE
--------------------------------------------------------------------------------
- Semakin tinggi mode yang dipilih (Turbo/Hardcore), semakin besar bobot reward 
  yang didapatkan per menit komputasi.
- Mode Hardcore mendapatkan bonus multiplier tambahan karena menyumbangkan 
  seluruh potensi hardware secara eksklusif.
