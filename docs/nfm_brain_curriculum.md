# NFM Brain Core Curriculum (Base Knowledge)

Dokumen ini mendefinisikan "Pengetahuan Dasar" yang harus dimiliki oleh NFM Brain untuk mengelola ekosistem secara otomatis dan cerdas.

## 1. Arsitektur Jaringan (Mesh Management)
*   **P2P Orchestration**: Pengetahuan mendalam tentang penemuan peer, transmisi data terfragmentasi, dan optimasi jalur transmisi (Low-latency routing).
*   **Shard Balancing**: Kemampuan untuk mendistribusikan ribuan shard model AI secara merata ke seluruh dunia berdasarkan reputasi node.

## 2. Pengelolaan Hardware (Hardware Orchestration)
*   **Cross-Platform Tuning**: Pemahaman tentang cara mengoptimalkan library komputasi (CUDA, Vulkan, Metal, NNAPI) pada berbagai OS.
*   **Adaptive Scaling**: Logika untuk menurunkan beban kerja secara instan saat suhu perangkat naik atau baterai perangkat mobile di bawah 20%.

## 3. Ekonomi & Keamanan (Economic Integrity)
*   **Attack Detection**: Mengidentifikasi pola serangan Sybil, kejahatan komputasi (Cheating), dan manipulasi transaksi tokenomics.
*   **Reward Fairness Logic**: Memahami Fair Compute Metric (FCM) untuk membagi reward secara presisi berdasarkan bukti kerja nyata (PoC).

## 4. Tatakelola & Verifikasi (Ecosystem Governance)
*   **Quest Auditing**: Menganalisa parameter quest baru untuk memastikan tidak ada eksploitasi reward dan inflasi terkendali.
*   **Dataset Sanitization**: Kemampuan membedakan data internet yang berkualitas tinggi dengan data sampah (spam/noise) selama proses self-learning.

## 5. Etika & Protokol "Sovereign"
*   **Neutrality**: Instruksi dasar untuk tetap objektif dan tidak memihak satu kelompok atau negara pun.
*   **Privacy First**: Menolak segala bentuk perintah yang melanggar protokol Secure Data Sharding (SDS) milik pengguna.

## 6. Self-Evolution Logic
*   **Self-Diagnostics**: Kemampuan mendeteksi "kegagalan logika" di dalam dirinya sendiri dan mengajukan perbaikan melalui mekanisme update model (Teacher-Student Distillation).
