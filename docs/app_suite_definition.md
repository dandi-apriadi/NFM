# NFM App Suite Definition

Daftar aplikasi dalam ekosistem NFM dengan detail fitur utama.

## 1. NFM Node Runner (The Engine)
*   **Fungsi**: Software untuk menyumbangkan hardware (CPU/GPU/NPU/RAM) ke jaringan.
*   **Zero-Config**: Instalasi satu-klik, deteksi hardware otomatis.
*   **Performance Modes**:
    *   **Quiet**: Penggunaan resource sangat rendah (<10%), tidak mengganggu pekerjaan lain.
    *   **Balanced**: Keseimbangan optimal (50%) untuk pemakaian harian.
    *   **Turbo**: Performa tinggi (80-90%) untuk user yang jarang memakai laptop/HP.
    *   **Hardcore**: 100% Resource didedikasikan untuk NFM. Reward maksimal.
*   **Mobile-Safe**: Mode "Only on Charge & Wi-Fi" untuk menjaga kesehatan baterai.

## 2. NFM Super-App (Chat & Wallet)
*   **AI Chat**: Akses ke AI NFM (Teks, Suara, Gambar).
*   **NFM-ID Wallet**: Kelola NFM Gold/Credit dengan biometrik (Bio-ZKP).
*   **Intelligent Transfer**: Kirim token via @alias dengan AI verification.
*   **In-App Swap**: Jual/beli token langsung menjadi fiat/stablecoin.
*   **Integrated Marketplace**: Akses langsung untuk beli/sewa "AI Skills" dan "Item Booster".
*   **NFM Drive**: Penyimpanan file terfragmen (SDS) secara personal.
*   **AI Autopilot**: Perintah AI untuk melakukan tugas di web (Searching, Summarizing, Booking).
*   **Local Knowledge Graph**: "Second Brain" AI yang belajar dari dokumen pribadi user secara aman.
*   **Item Inventory**: Lihat koleksi item, status segel, dan aktivasi boost.

*   **NFM Web Portal (Explorer & Governance)**: Akses statistik, voting, dan **NFM Drive Web Viewer**.
*   **Web-to-Mesh Layer**: Akses statistik dan voting dari browser melalui NFM Edge Gateway.
*   **Decentralized Hosting**: Website NFM di-host oleh jaringan NFM sendiri (Anti-Block) sebagai backup jika domain utama terkendala.
*   **Public RPC Nodes**: Menyediakan jalur data terbuka bagi developer pihak ketiga untuk membangun "Community Explorer" mereka sendiri. Data bersifat transparan dan dapat ditarik oleh siapa saja.
*   **NFM-ID Bridge (Secure Login)**: Login aman di web menggunakan scan **QR Code** dari HP. Verifikasi biometrik tetap dilakukan di HP, sementara sesi terbuka di browser.
*   **Governance & Staking**: Interface layar lebar untuk membaca proposal AI yang kompleks.

## 4. NFM Developer Portal
*   **API Management**: Kontrol akses profesional untuk integrasi pihak ketiga.
*   **Marketplace**: Jual beli AI Agent ("Skill").
*   **Admin Panel**: Khusus Founder untuk manajemen API Key Payment (Xendit).

## 5. NFM CLI Tool
*   **Professional Tool**: Untuk developer melakukan debugging dan deployment model.
## 6. NFM-ID Bridge Protocol
*   **Cross-Device Auth**: Protokol yang menghubungkan keamanan biometrik HP dengan kemudahan akses Web Browser.
*   **Session Management**: Token akses didekripsi secara lokal di HP sebelum diberikan ke browser untuk menjaga privasi "Zero-Trust".
