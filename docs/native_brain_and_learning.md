# NFM Native Brain & Distributed Learning

Dokumen ini menjelaskan mekanisme kecerdasan bawaan NFM yang dapat tumbuh secara kolektif.

## 1. Native NFM Brain (The Core Intelligence)
*   **Model Inti**: NFM memiliki satu model AI "Sovereign" yang menjadi bagian dari protokol dasar.
*   **Bukan Milik Individu**: Berbeda dengan "Skill" di Marketplace, Native Brain dikelola sepenuhnya oleh konsensus jaringan.
*   **Akses Gratis**: Dirancang sebagai utilitas publik dasar bagi seluruh pemegang NFM-ID.

## 2. Jendela Pembelajaran (Learning Window)
NFM tidak statis; ia dapat mempelajari pengetahuan baru melalui instruksi DAO:
*   **Proposal Kurikulum**: Pemegang token (terutama pemegang besar/Founder) mengajukan topik pembelajaran baru.
*   **Dataset Sourcing**: Proposal mencakup sumber data (Dataset URL atau Instruksi Crawling).
*   **Voting Approval**: Jika disetujui, jendela pembelajaran dibuka selama periode tertentu (Epoch).

## 3. Distributed Training / Fine-tuning (PoUW)
Saat jendela pembelajaran terbuka, seluruh node di jaringan beralih fungsi menjadi "Pelajar":
*   **SDS-Compliant Data Sharding**: Dataset pembelajaran dipecah menggunakan protokol **Secure Data Sharding (SDS)**. Setiap node hanya melihat fragmen data yang tidak terbaca tanpa konteks utuh, menjaga privasi sumber dataset.
*   **Local Gradient Computation**: Laptop dan HP melakukan komputasi pembelajaran kecil di background.
*   **Federated Aggregation**: Hasil pembelajaran dari jutaan node digabungkan kembali di cluster Genesis untuk memperbarui bobot Native Brain secara permanen.

## 4. Internet-Enabled Nodes (Real-time Ingestion)
*   **Sandboxed Crawling**: Node terpilih (berdasarkan reputasi & mode) diberikan ijin untuk mengakses internet guna mengambil data spesifik sesuai instruksi voting.
*   **Verification Mesh**: Data yang diambil diverifikasi oleh node lain untuk memastikan tidak ada "berita palsu" atau "data sampah" yang masuk ke otak AI.

## 5. Ekonomi Pembelajaran
*   **Learn-to-Earn**: Node yang mengalokasikan resource untuk melatih Native Brain mendapatkan multiplier reward lebih besar daripada hanya melakukan inferensi chat biasa.
*   **Stake-Weighted Curriculum**: Semakin banyak token yang di-stake pada suatu proposal belajar, semakin tinggi prioritas komputasi yang diberikan jaringan untuk topik tersebut.
