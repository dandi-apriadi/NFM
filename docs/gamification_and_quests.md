# NFM Quest, Syndicates, and Item Economy (v7.7)

## 1. NFM Quest (Misi & Tantangan)
- **[Awal] The Newcomer**: Selesaikan 10 task pertama Anda. 
    - **Reward**: 1.000 NVCoin.
    - **Kuota**: **Hanya untuk 20.000 pendaftar pertama.** (Quest otomatis hilang setelah kuota terpenuhi).
- **[Awal] The First Fragment**: Selesaikan komputasi shard pertama Anda. Reward: Badge "Pioneer".
- **[HARD/LEGENDARY] Eternal Guardian**: Node nonstop 30 hari. Reward: **Legacy Core**.

## 2. Mystery Box & VIP Items
- **Mekanisme**: Didapat dari **AI Feedback Valid**.
- **Staking**: Maksimal 3 item total di-stake per akun.

## 3. Aturan Keamanan & Sybil Defense [NEW v7.7]
Untuk mencegah manipulasi reward ("Farming"), NFM menerapkan aturan:
- **1 IP = 1 Account (Initial)**: Setiap alamat IP jaringan hanya diizinkan melakukan 1 proses pendaftaran akun dasar.
- **Verification Waterfall**: Jika sistem mendeteksi IP/Device yang sudah terdaftar, pengguna harus membuktikan kepemilikan melalui:
    1. **Password** (Akses Cepat).
    2. **Biometrics** (NFM Brain Signature).
    3. **Private Key / Seed Phrase** (Akses Blockchain).
    4. **Device ID Binding** (Verifikasi Perangkat Terdaftar).
