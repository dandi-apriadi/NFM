# NFM Quest, Syndicates, and Item Economy (Master v7.9.1)

Sistem ini dirancang untuk memberikan penghargaan tertinggi bagi dedikasi, kejujuran, dan kontribusi nyata.

## 1. NFM Quest (Misi & Tantangan)
- **[Awal] The Newcomer**: Selesaikan 10 task pertama Anda. 
    - **Reward**: 1.000 NVCoin.
    - **Kuota**: **Hanya untuk 20.000 pendaftar pertama.**
- **[Awal] The First Fragment**: Selesaikan komputasi shard pertama Anda. Reward: Badge "Pioneer".
- **[HARD/LEGENDARY] Eternal Guardian**: Node nonstop 30 hari. Reward: **Legacy Core**.

## 5. Mystery Box System
- **Mekanisme**: Didapat dari **AI Feedback Valid**.
- **Fitur**: Memberikan item random berdasarkan persentase kelangkaan global.

## 6. Aturan Keamanan & Sybil Defense
- **1 IP = 1 Account**: Mencegah pendaftaran massal.
- **Verification Waterfall**: Password -> Biometrics -> Private Key -> Device ID.
- **Slashing/Burning**: Pelanggaran membakar item staked dan mengembalikannya ke Mystery Box Pool.
