# NFM-ID Identity & Title System (v7.0)

Sistem identitas terpadu untuk memberikan prestise dan peran dalam jaringan NFM.

## 1. Administrative Titles
- **Founder**: Akses tertinggi pada tata kelola awal.
- **Co-Founder**: Peran pendukung strategis.
- **Dev Team**: Akses ke repositori pengembangan.

## 2. Brain-Verified (KYC)
- **Verified User**: Berdasarkan Biometrik & Identitas legal.
- **Corporate**: Untuk entitas bisnis/perusahaan.

## 3. Achievement Titles
- **Guardian**: Pemegang item Legacy Core.
- **Early Bird**: Terdaftar dalam 10.000 NFM-ID pertama.
- **Apex Workhorse**: Kontributor terbaik selama 1 tahun.

## 4. NFM-ID Marketplace
Username (@handle) adalah aset digital yang dapat diperjualbelikan atau 
ditransfer ke ID unik nfm_... lainnya.

## 5. Biaya Transaksi
- **Transfer/Sale Tax**: Pajak rendah dialokasikan ke porsi Founder / Core Dev.
