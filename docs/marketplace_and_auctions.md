# NFM Universal Marketplace & Auction System (v7.0)

Sistem perdagangan aset digital (NFM-ID, Item Boost, NFT, Sharded Data).

## 1. Tipe Lelang
- **English Auction**: Penawar tertinggi menang.
- **Dutch Auction**: Harga turun secara berkala.
- **Sealed-Bid**: Penawaran tersembunyi.

## 2. Fitur Keamanan
- **Escrow**: NVCoin dikunci selama masa penawaran.
- **Anti-Sniping**: Perpanjangan waktu otomatis di menit terakhir.

## 3. Alokasi Pendapatan
- Successful Sale Tax (5%) masuk 100% ke porsi **Founder / Core Dev**.
