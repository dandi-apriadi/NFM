🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

Via UI:
  1. Open Settings (⚙️ icon)
  2. Click "Preferences" tab
  3. Scroll to "Development Tools" section
  4. Click "Reset Blockchain to Genesis" (red button)
  5. Enter your API secret
  6. Confirm reset
  7. System resets and page auto-reloads

Via Test Script:
  powershell -ExecutionPolicy Bypass -File .\test_blockchain_reset.ps1

Via Direct API Call:
  POST http://127.0.0.1:3000/api/admin/reset
  Header: X-NFM-Signature: <HMAC-SHA256>
  Body: {}
